# MBA Strategy & Executive Presentations

This section contains selected MBA strategy projects completed through Nyenrode Business Universiteit.

The main PDF in this folder is a visual project summary. It includes photos and short descriptions from selected company-based presentations and MBA project moments. It is intended to provide public-facing evidence of project exposure and presentation settings, not the full confidential slide decks or detailed analysis behind each project.

Detailed project materials can be shared separately upon request where appropriate.
